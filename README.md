# py-snips

This repository documents commonly used or easily forgotten Python functions. These example functions are useful when LLMs are not allowed or available, or when search engines are too slow.

## Examples

- Imports: `test_import`
- JSON: `load_json`, `load_large_json`, `json_dump_vs_dumps`, `json_load_vs_loads`, `write_json`
- Filesystem globbing: `glob_dir`, `glob_dir_recur`
- Built-ins and collections: `isinstance_example`, `intenum_example`, `named_tuple_example`, `deque_example`
- Logging: `logging_example`
- YAML: `write_yaml`, `load_yaml`
- Bytes and hashing: `crc_data`, `hexdump`, `hash_data`, `xor`
- Compression: `gzip_*`, `zip_*`
- Binary parsing: `cytypes_math`, `cytype_enum`, `read_value_little_endian`
- YARA: `scan_buffer_yara`
